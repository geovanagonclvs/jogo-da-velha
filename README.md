### Variávels Globais
Iniciei criando as váriavés que não podem ser apagadas depois das jogadas, pois servem como a "memória" do jogo da velha.

- 'let tabuleiro=[]': cria o nosso array que irá armazenar nossa matriz 3x3
- 'const jogador1=1': numera o jogador 1 como 1
- 'const jogador2=2': numero o jogador 2 como 2

**obs**: usei a palavra-chave **const** para declarar nosssa variável pois esses valores serão fixos e não muradão conforme as jogadas

## Criação do Tabuleiro Lógico
Para criar um tabuleiro 3x3 no `JS`, é preciso criar uma estrutura chamada de **Matriz** ( um aray que contém outro array dentro). Ela irá conter 2 dimensões: a primeira (que será o nosso primeiro array) irá representar as linhas, e a segunda dimensão ( o nosso segundo array) irá representar as colunas de cada linha.
